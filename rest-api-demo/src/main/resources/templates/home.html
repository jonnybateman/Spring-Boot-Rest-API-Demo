User:

Role(s): STUDENT INSTRUCTOR ADMIN



Depending on the role of the user various links are displayed in the NavBar. These links will permit access to the database and display relevant information. Only information authorised according to the user's role will be displayed. When registering a new user the individual can create a user with either the STUDENT or INSTRUCTOR role. An instructor will also have the STUDENT role privileges.

Here the database is tracking courses, the instructors leading the courses, and the students taking the courses. As well as this students are also able to leave reviews on particular courses.
